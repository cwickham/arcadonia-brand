# arcadonia-brand

Brand assets and `_brand.yml` for the **Arcadonia Health Department** — a fictional public-health agency used in Quarto branding/accessibility demos.

## Contents

- `_brand.yml` — colors, typography, and logo definitions
- `images/ahd-seal.svg` — agency seal
- `images/asr-wordmark.svg` — Arcadonia Surveillance Reports wordmark

## Usage

Consume from another Quarto project via [`quarto use brand`](https://quarto.org/docs/authoring/brand.html):

```bash
quarto use brand <path-or-url-to-this-repo>
```
